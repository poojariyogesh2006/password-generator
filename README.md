# password-generator
🔐 Advanced Password Generator

A secure and user-friendly Password Generator built with Python and Tkinter that helps users create strong, random, and highly secure passwords. The application follows modern password security practices by ensuring the inclusion of uppercase letters, lowercase letters, numbers, and special characters while validating password strength and security rules.

The project provides an intuitive graphical user interface (GUI) where users can customize password length, choose character types, exclude confusing characters, and instantly generate passwords suitable for personal, professional, and enterprise-level security requirements.

✨ Features
🔐 Generate strong and secure random passwords
🔤 Supports uppercase and lowercase letters
🔢 Includes numbers and digits
⚡ Includes special characters and symbols
📏 Custom password length selection
🚫 Option to exclude confusing characters (O, 0, I, l, 1)
🛡️ Built-in password security validation
📊 Password strength indicator (Medium, Strong, Very Strong)
✅ Security status verification
📋 One-click copy to clipboard
🎨 Simple and easy-to-use graphical interface
🚀 Fast and lightweight desktop application
🛠️ Technologies Used
Python
Tkinter (GUI)
Random Module
String Module
🔒 Security Rules Implemented
Minimum password length of 8 characters
At least one uppercase letter (A-Z)
At least one lowercase letter (a-z)
At least one numeric digit (0-9)
At least one special character (!, @, #, $, %, etc.)
Prevents consecutive repeated characters
Password strength validation system
Optional exclusion of ambiguous characters
🚀 How It Works
Enter the desired password length.
Select the character categories to include.
Optionally exclude confusing characters.
Click Generate Password.
The application creates a secure password that satisfies all security requirements.
Review the strength rating and security status.
Copy the generated password directly to your clipboard.
